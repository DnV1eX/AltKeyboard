Main Features

Keyboard Overlay

The overlay appears when you hold custom modifier keys. You can also show it permanently using the app menu or by pressing the Option key twice. Drag the overlay by its edges to reposition the keyboard.

  • Highlights pressed keys and shows tooltips on hover.
  • Supports mouse clicks or physical key presses.
  • Modifiers stay pressed when clicked until a non-modifier key is pressed.

Virtual Layouts

Create custom layouts in the Virtual Layout Editor and manage them in Layouts Settings:

  • Customize: Add keys with custom input, legends, and hints.
  • Activate: Set layout activation modifiers (disable them in the menu or by pressing the Option key three times).
  • Group: Assign layouts to groups, switched with function keys (F1-F12) or the app menu.
  • Share: Export layouts as .alt files or import from others.

Customization

  • Overlay: Set the overlay display delay, choose modifier keys, hide key names, labels, or hints.
  • Appearance: Adjust scale, material transparency, and light/dark mode.
  • Extra: Enable key press or clipboard sounds, clipboard notifications, and more.
